# Questions

You are given an array of integers, and your task is to find the first element that appears more than once in the array. Implement a function find_first_duplicate(arr) that returns:

the first duplicate element or -1 if there are no duplicates.

Examples


```
Input: [1,3,4,2,2,5,6]
Output: 2

Input: [4,5,2,5,3]
Output: 5
```
